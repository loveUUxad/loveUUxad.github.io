后端
  • java

最近公司新开发的一个项目中，我们使用的的springboot+tkmybatis来进行开发，之前没了解到tkmybatis，于是去查阅了一些资料，算是整理一下自己的学习笔记吧

什么是tkmybatis

tkmybatis是一个在mybatis的基础上进一步封装的一个业务层框架，它省去了之前mybatis中需要大量编写sql的弊端,大大简化了我们的开发。
